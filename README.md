# 🧮 Modern Calculator

A modern and responsive calculator web application developed using HTML, CSS, and JavaScript. This project was created as **Task 3** for the **CodSoft Web Development Internship**.

## 🚀 Live Demo

https://amitkumar-7393.github.io/CODSOFT-Calculator.App/

## ✨ Features

- Responsive Design
- Modern Glassmorphism UI
- Basic Arithmetic Operations
- Keyboard Support
- Delete (DEL) Function
- All Clear (AC)
- Percentage Calculation
- Error Handling
- Dark/Light Theme Toggle
- Mobile Friendly
- Clean User Interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
